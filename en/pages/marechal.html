Marshal

About the event

The Marshal event challenges players to complete missions and battles using specific formations. Scoring depends on combat performance and completing daily objectives.

How to prepare:

  • Build balanced formations with tanks, aircraft and missiles
  • Invest in heroes recommended for the event battle type
  • Complete daily missions so you don't lose accumulated points
  • Check with the alliance for the best troop compositions
